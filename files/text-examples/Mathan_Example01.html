Вычислить \begin{equation} \mathop {\lim }\limits_{x \to 1} \frac{{{x^4} - 1}}{{{x^3} - 4{x^2} + 3x}}. \end{equation} Ответ дать с точностью до 2-го знака после запятой.

1. Подставим предельное значение в исследуемую функцию и определим вид неопределенности. Подстановка предельного значения $x = 1$ в функцию \begin{equation} f(x) = \frac{{{x^4} - 1}}{{{x^3} - 4{x^2} + 3x}} \end{equation} дает неопределенность $\left\{ \dfrac{0}{0} \right\}$.
$f(x)$ — дробно-рациональная функция, значение $x = 1$ является корнем многочленов, стоящих в числителе и знаменателе, следовательно, эти многочлены без остатка делятся на $x – 1$. 2. Выделим в многочленах, стоящих в числителе и знаменателе, множитель $x - 1$: $${x^4} - 1 = ({x^2} - 1)({x^2} + 1) = (x - 1)(x + 1)({x^2} + 1),$$ $${x^3} - 4{x^2} + 3x = x({x^2} - 4x + 3) = x(x - 1)(x + 3).$$ 3. Подставим преобразованные многочлены в функцию $f(x)$ $(2)$: $$\frac{{{x^4} - 1}}{{{x^3} - 4{x^2} + 3x}} = \frac{{(x - 1)(x + 1)({x^2} + 1)}}{{x(x - 1)(x - 3)}}$$ 4. Разделим числитель и знаменатель на общий множитель $x - 1$: $$\frac{{(x - 1)(x + 1)({x^2} + 1)}}{{x(x - 1)(x - 3)}} = \frac{{(x + 1)({x^2} + 1)}}{{x(x - 3)}}$$ 5. Вычислим предел от преобразованной исследуемой функции $f(x) = \dfrac{{(x + 1)({x^2} + 1)}}{{x(x - 3)}}$: $$\mathop {\lim }\limits_{x \to 1} \frac{{(x + 1)({x^2} + 1)}}{{x(x - 3)}} = \frac{{2 \cdot 2}}{{1 \cdot ( - 2)}} = - 2$$ Таким образом, $$\mathop {\lim }\limits_{x \to 1} \frac{{{x^4} - 1}}{{{x^3} - 4{x^2} + 3x}} = -2.$$
